Aufgabe

Die Basisdatentypen für ganze Zahlen (int, long, etc.) sind beschränkt - was also tun, wenn Zahlen verarbeitet werden sollen, deren Größe den Wertebereich der Basisdatentypen übersteigt?

Eine Möglichkeit besteht darin, direkt auf einer String-Repräsentation der Zahlen zu arbeiten. Schreiben Sie ein Programm, das zwei nicht-negative ganze Zahlen in Binärdarstellung einliest und deren Summe in Binärdarstellung ausgibt.

Zur Vereinfachung können Sie dabei annehmen, dass die einzulesenden Strings höchstens die Länge 100 haben. Die Antwort soll keine führenden Nullen enthalten. Ist das Ergebnis Null, so soll eine einzelne '0' ausgegeben werden.

Input

 Die Eingabe erfolgt über die Standardeingabe und besteht aus zwei Zeilen. Jede Zeile besteht aus einer Folge von '0' und '1' der Länge <= 100 und ist mit Newline abgeschlossen.
Beispiel 1 
  000000
  0
Beispiel 2 
  111000111000
  111111111111

Output

 Die Ausgabe enthält eine Zeile, nämlich die Summe der beiden Eingaben in Binärdarstellung ohne führende Nullen. Ist das Ergebnis Null, so soll eine einzelne '0' ausgegeben werden. Die Zeile wird durch ein Newline abgeschlossen.
Beispiel 1 
  0
Beispiel 2 
  1111000110111
