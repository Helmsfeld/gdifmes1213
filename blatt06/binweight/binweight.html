Aufgabe

Wir definieren das binäre Gewicht w2 einer natürlichen Zahl als die Anzahl der 1-en in ihrer Binärdarstellung. So ist etwa w2(4) = 1 oder w2(7) = 3. Implementieren Sie ein Programm binweight.c, das zu einer gegeben natürlichen Zahl ihr binäres Gewicht berechnet. Als zulässigen Definitionsbereich dürfen Sie unsigned int annehmen.

Input

 Die Eingabe erfolgt per stdin und besteht aus einer oder mehreren Zeilen. Jede Zeile enthält eine ganze, nichtnegative Zahl in Dezimaldarstellung. Die Eingabe wird durch EOF abgeschlossen.
Beispiel 
  4
  7

Output

 Geben Sie für jede Eingabezeile eine Zeile aus, die das binäre Gewicht der entsprechenden Zahl in Dezimaldarstellung enthält. Schließen Sie jede ausgegebene Zeile durch ein Newline ab.
Beispiel 
  1
  3
