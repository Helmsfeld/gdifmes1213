Aufgabe

Erweitern Sie Ihr Programm prim.c von Blatt 5 zu einem Programm nextprim.c, welches zu einer gegebenen Zahl n die kleinste Primzahl p mit p > n bestimmt.

Input

 Die Eingabe erfolgt per stdin und besteht aus einer oder mehreren Zeilen. Jede Zeile enthält genau eine ganze, nichtnegative Zahl in Dezimaldarstellung. Die Eingabe wird durch EOF abgeschlossen. Sie dürfen annehmen, dass die nächste gesuchte Primzahl stets im Wertebereich von unsigned long int liegt.
Beispiel 
  0
  7

Output

 Geben Sie für jede Eingabezeile eine Zeile aus, welche die entsprechende nächstgrößere Primzahl enthält. Schließen Sie jede ausgegebene Zeile durch ein Newline ab.
Beispiel 
  2
  11
