Aufgabe

Implementieren Sie ein Programm prim.c, welches zeilenweise Integerzahlen einliest und prüft, ob diese eine Primzahl darstellen. Schreiben Sie dazu eine Funktion isprime() in ihrem Programm welche eine einzelne Zahl testet. Abhängig vom Rückgabewert der Funktion isprime() soll das Programm prim! oder nicht prim! ausgeben.

Input

Die Eingabe erfolgt per stdin und besteht aus einer oder mehreren Zeilen. Jede Zeile enthält eine ganze, nicht-negative Zahl in Dezimaldarstellung. Die Eingabe wird durch EOF abgeschlossen.
Beispiel 

  2
  0
  199999991

Output

Geben Sie für jede Eingabe-Zeile eine Ausgabe-Zeile mit prim! oder nicht prim! (gefolgt von einem Newline) aus, je nachdem, ob die entsprechende Zahl der Eingabe prim ist oder nicht.
Beispiel 

  prim!
  nicht prim!
  prim!
