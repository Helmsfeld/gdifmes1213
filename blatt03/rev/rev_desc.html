Aufgabe

Implementieren Sie eine Funktion 

    void reverse(const char *s, char *r)
die den String s in umgekehrter Reihenfolge in den String r kopiert. Sie dürfen dabei annehmen, dass r alle Zeichen aufnehmen kann. Die main-Funktion soll eine oder mehrere Zeilen von der Standardeingabe lesen und jede gelesene Zeile umgekehrt ausgeben. Leere Zeilen sollen dabei nicht weggelassen werden.

Input

 Die Eingabe besteht aus einer oder mehreren Zeilen, jede Zeile inklusive Newline ist maximal 80 Zeichen lang. Beispiel-Eingabe: 
  abcde
  ab01bb

  otto

Output

 Die Ausgabe enthält jede Zeile in umgekehrter Reihenfolge. Jede Zeile wird durch ein Newline abgeschlossen. Beispiel-Ausgabe: 
  edcba
  bb10ba

  otto
